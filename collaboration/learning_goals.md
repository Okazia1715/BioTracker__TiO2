# Learning Goals

The project has been designed with the goal of realizing something
more than just a useful tool, but also a way to develop technical,
scientific, and integrative skills that combine the expertise of
deep learning and biomedical image analysis.

## Training and Customization of Cellpose Models

The main goal is to understand how the Cellpose model works beyond
its default pretrained version. This requires preparing the data
for training, making masks, and then training a model that is
specific for colon histology images. The goal is to move from
using a pre-existing model to developing and validating a solution
for image segmentation specific to a domain.

## Application of Cutting-Edge Deep Learning to an Open Scientific Question

The project has been designed with the goal of applying cutting-edge
image segmentation tools to a scientifically interesting and
outstanding question: the quantitative localization of nanoparticles
in tissue sections. The goal is not only to implement a solution
but also integrate it into a structured analysis that has the
potential for producing scientific outputs.

## Construction of Independent Scientific Software

One of the major learning objectives is to develop a complete
scientific process as software for standalone use. This involves
developing a graphical user interface, an organized process
pipeline, file architecture, output generation, and reproducibility.
The key aspect is to transform an algorithm into a scientific
software package.

## Integration of Segmentation, Simulation, and Quantification

The software involves integrating multiple components that are
normally treated independently. These components include image
preprocessing, deep learning segmentation, synthetic nanoparticle
simulation, spatial localization analysis, and statistical output
generation. The learning objective is to understand the
interrelation between the components and create an integrated
analytical software package rather than a set of individual scripts.

## Reproducibility and Scientific Transparency

The other learning objective is to incorporate open science
principles into the software. This involves data integrity, output
organization, detailed documentation of the process, and
reproducibility of the results. The learning outcome involves not
just technical knowledge but also scientific transparency.
